    Background Subtraction

    (Delhi, India)

    A robust video surveillance system does not depend only on the careful placements of cameras but is also robust to whatever is in its visual field or whatever lighting effects occur.

    A standard method is effective in situations where objects move continuously and the background is visible a significant portion of the time, it is not robust to scenes with many moving objects particularly if they move slowly. It also cannot handle bimodal backgrounds, recovers slowly when the background is uncovered and has a single, predetermined threshold for the entire scene. )

    Durgapur, India

    Human_Activity_Recognition Using Seismic Sensor and Learning Techniques Human Activity Recignition has been an important topic in research community since the last decade. In this work, we propose a novel METHOD that can be used for passive human activity classification leveraging the geophones, signal processing and ensemble learning techniques.

Micro seismic event detection using seismic sensor and machine learning

Durgapur, India

The project is divided into following parts: Record datasets in single geophones using MC 1608FS-plus of no movement and movement Apply Wavelet based thresodling using hard thresolding . Extract 5 features from the manipulated datasets (mean,median,standard-deviation,variance...), Fit into learning algorithms like Random Forest, Support Vector Machine (linear/ rbf ), Decision Tree, ANN, KNN etc. Classifiers achieves highest accuracy of 96% between no movement and movement
